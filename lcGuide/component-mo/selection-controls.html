Principle

콘텐츠 내에서 옵션 선택 시 사용 되는 유형별 컨포넌트 입니다.
체크박스는 기본적으로 우측에 위치하는것을 기본으로 하며 콘텐츠 안에 종속으로 사용하는
경우 예외적으로 좌측에 위치됩니다.
라디오버튼은 기본적으로 우측에 위치하는것을 기본으로 하며 2종류의 컴포넌트 사용시
예외적으로 좌측에 위치됩니다.

Types
Check Box링크

  • 설정에 대한 완료 전, 사용자 확인이 필요한 경우, 여러 개의 항목 중 다중 선택이 필요할 경우 사용하며 팝업 내에 사용 시 공통 컬러가 적용되며 자세한 사항은 공통 컬러 가이드를 참조해 주세요.
Radio Button링크

  • Default로 첫 번째 항목이 선택되어 있음, 여러 개의 항목 중 단일 선택이 필요할 경우 사용하며 팝업 내에 사용 시 공통 컬러가 적용되며 자세한 사항은 공통 컬러 가이드를 참조해 주세요.
Toggle Switch링크

  • 물리적 스위치 모습으로 주로 켜기/끄기 용도로 사용하며 팝업 내에 사용 시 공통 컬러가 적용되며 자세한 사항은 공통 컬러 가이드를 참조해 주세요.
Select Box링크

  • 한눈에 보여질 필요가 없는 옵션 집합에서 하나의 옵션을 선택해야 할 경우 사용합니다.
