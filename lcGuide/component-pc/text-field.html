Text Field

Principle

사용자가 문자와 숫자 타입의 텍스트를 입력할 수 있는 컴포넌트입니다.
Placeholder 문구는 사용자의 입력값이 들어오면 문구는 사라지며, 필요 시 상단에 병기하도록 합니다.
텍스트 필드의 사용 목적에 따라 아이콘, 클리어 버튼, 도움말, 카운트는 생략 가능합니다.

Types
Single Line링크

  • 텍스트가 인풋박스 영역보다 길어질 때, Text가 좌측으로 숨겨지면서 입력됩니다.
  • 최대 입력가능한 글자수, 용량(byte)이 차면 더 이상 입력이 수행되지 않으며, 필요에 따라 조정 가능합니다.
Multi Line링크

  • 인풋 텍스트를 두 줄 이상으로 노출 할때 사용합니다.
  • 텍스트가 인풋박스 영역보다 길어질 때, Text를 다음행으로 줄바꿈하여 한 줄씩 아래로 늘어나며 입력됩니다.
  • 화면에 노출하는 줄바꿈 행의 최대 행 수는 화면에 적합하게 정의 해야 합니다.
  • 최대 입력가능한 글자수, 용량(byte)이 차면 더 이상 입력이 수행되지 않으며, 필요에 따라 조정 가능합니다.
Multi Box링크

  • 텍스트 삭제 버튼과 스크롤이 같이 나올 때 사용합니다.
  • 레이어 팝업, 과정내 작성에서는 공통컬러를 사용합니다. (Typing = #5F7AD1)
  • 텍스트가 인풋박스 영역보다 길어질 때, Text를 다음행으로 줄바꿈하여 한 줄씩 아래로 늘어나며 입력됩니다.
  • 최대 입력가능한 글자수, 용량(byte)이 차면 더 이상 입력이 수행되지 않으며, 필요에 따라 조정 가능합니다.
  • 화면 노출되는 최대 행수가 넘는 입력값을 받을 수 있는 경우, 스크롤 생성 됩니다.
  • 텍스트가 기록되면 삭제 버튼이 생성 됩니다.
