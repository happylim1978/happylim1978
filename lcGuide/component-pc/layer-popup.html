Principle

사용자에게 중요한 정보 또는 페이지 전환 없이 정보를 보여줄 때 사용합니다.
중요 정보를 포함할 수 있으며, 하단 최우측에 해당 팝업의 주 목적버튼을 배치합니다.
타이틀 영역은 생략 가능 하며 팝업의 목적을 한번 더 상기시켜 강조할 필요가 있는 경우에 사용합니다.
방해적인 요소가 될 수 있으므로, 자제하여 사용합니다.
팝업 사이즈(가로) : 450 / 520 / 700 / 740 / 790(기본) / 800 / 900 / 1164 / 1224px
팝업 최대 높이값은 658px이며, 스크롤은 콘텐츠 영역이 558px 이상일때 생성됩니다.
