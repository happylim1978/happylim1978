Principle

버튼은 사용자가 작업을 수행하거나 다른 페이지로 이동할 수 있게 하는 컴포넌트입니다.
버튼은 사용처에 따라 적합한 유형으로 분류 사용됨을 원칙으로 하며,
배치상황에 따라 너비는 유동적으로 적용 될 수 있습니다.

Types
Box Button : Text링크

  • 박스형의 가장 기본이 되는 텍스트 형태로 솔리드와 라인으로 구분하며,
    텍스트레이블과 백그라운드로 구성되어 있습니다.
Box Button : Icon+Text링크

  • 박스의 형태에 아이콘과 텍스트가 혼합형으로 나오는 버튼을 의미합니다.
Keyword Button링크

  • 키워드와 관련된 버튼 형태로 관련키워드, 추천키워드, 필터형 버튼으로 구분됩니다.
Text Button : Line링크

  • 텍스트 형태 버튼으로 오버시 백그라운드 박스형태로 표현되거나 밑줄 액션을 적용하여
    일반 정보와 구분되어 인지할 수 있어야 합니다.
Text Button : Link링크

  • 텍스트에 링크가 적용 되어 있는 버튼, 비교적 중요도가 낮거나
    버튼이 시각적으로 방해 되는 경우 사용됩니다.
Icon Button링크

  • 아이콘 형태 버튼으로 아이콘에 Bg 액션을 적용하여 일반 정보 아이콘과 구분될 수 있어야 합니다.
  • 아이콘 버튼의 유형은 기본이 되는 라인 아이콘과 솔리드아이콘으로 분류될 수 있습니다.
Common Button링크

  • 컬러, 사이즈 등 모든 페이지에서 공통을 적용 되는 버튼을 의미합니다.
  • 공통버튼의 유형은 크게 동영상 관련 버튼과 스와이프 버튼으로 분류됩니다.
Function Button링크

  • 기능버튼은 콘텐츠에 종속된 기능적인 역활을 수행하는 버튼입니다.
  • 박스형과 아이콘과 텍스트 조합형으로 분류되어 사용됨을 원칙으로 합니다.
